Auto-atribuição de Vendedor: ao criar ou importar um cliente, o sistema verifica o campo uf e associa automaticamente o vendedor que cobre aquele estado. Se nenhum vendedor cobrir a UF, o campo fica nulo.
Importação de Catálogo por IA: a rota POST /catalogs/:id/import-pdf envia o arquivo PDF ao Claude (claude-sonnet-4-6). O modelo lê o catálogo e retorna os produtos com todos os campos de especificação técnica como JSON, que são inseridos automaticamente no banco. Requer ANTHROPIC_API_KEY.

RoutesDefinição de endpoints, configuração do Multer para uploads
ControllersLógica de requisição/resposta HTTP, tratamento de erros amigáveis

Instalação e Execução Local

1. Clonar e instalar dependências

bun install
cd server && bun install
cd ../client && bun install

2. Configurar variáveis de ambiente

cp .env.example server/.env

Conteúdo do server/.env:

DATABASE_URL=postgresql://user:password@ep-xxx.neon.tech/neondb?sslmode=require
PORT=8000
NODE_ENV=development
ANTHROPIC_API_KEY=sk-ant-...   # necessário para import de catálogos por PDF

3. Criar as tabelas no banco

Acesse o SQL Editor do Neon.tech e execute:

server/migrations/001_schema.sql
server/migrations/002_add_tipo_to_products.sql   (se necessário)
server/migrations/003_add_fields_to_clients.sql  (se necessário)

4. Rodar em desenvolvimento

bun run dev
ServiçoURL
Frontendhttp://localhost:5173
Backendhttp://localhost:8000

5. Debug no VS Code (F5)

O projeto inclui .vscode/launch.json com a configuração "Debug Backend".

  1. Abra a pasta raiz no VS Code
  2. Pressione F5 ou vá em Run → Start Debugging
  3. Selecione "Debug Backend"
  4. O servidor inicia na porta 8000 com debugger ativo
As variáveis de ambiente são carregadas automaticamente de server/.env. Breakpoints funcionam em todos os arquivos de server/src/.

Deploy no Render

Root Directoryserver

Root Directoryclient
Build Commandbun install && bun run build
Publish Directorydist

Variável: VITE_API_URL = URL do backend no Render

Use o render.yaml na raiz para deploy automático dos dois serviços de uma vez.
